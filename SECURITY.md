# Security Policy

## Supported Versions

This project does not provide dedicated security updates for specific versions.
However, identified security issues may be addressed at the maintainer’s discretion.

| Version | Supported         |
| ------- | ----------------- |
| All     | :warning: Limited |

## Reporting a Vulnerability

If you discover a security issue, please report it by opening an issue in the repository.

Include:

* A clear description of the vulnerability
* Steps to reproduce (if applicable)
* Potential impact

Reports will be reviewed on a best-effort basis. If the issue is confirmed, a fix may be implemented, but no guarantees are made regarding timelines or patch releases.
